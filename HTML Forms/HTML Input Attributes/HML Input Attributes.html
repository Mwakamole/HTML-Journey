The Size Attribute

Specifies the visible width, in characters of an input field.
The default value for size is 20. In the example beleow, the size of the 'pin' input field has been set to four.


The min and max Attributes

Used for specifying the maximum and minimm values of an input field.
It works with number, range, date, time, month and week input types In the esxample below, the minimum value for the range(slider) has been set to 0 and the maximum value set to 50

The Multiple Attribute

Specifies that a user is allowed to input more than one value of an input
Works with the email and file input types

The required Attribute

Specifies that an input field must be filled out before submission




The step Attribute

Specifies the intervals of an input field

The autofocus Attribute

Specifies that an input field should automatically focus when the page loads
